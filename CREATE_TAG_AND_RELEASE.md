# Инструкция по созданию тега и релиза

## Статус сборки:
✅ Код успешно скомпилирован
✅ Файлы готовы в `release/win-unpacked/`

## Шаг 1: Создание установщика через Inno Setup

1. Откройте файл `installer.iss` в Inno Setup
2. Нажмите "Build" → "Compile" (или F9)
3. Установщик будет создан в папке `release/` с именем `SafeKey-Setup-1.0.0-x64.exe`

## Шаг 2: Создание тега версии

### Вариант A: Через GitHub веб-интерфейс (проще)

1. Перейдите на https://github.com/Actoric/SafeKey
2. Нажмите на "Releases" (справа в меню)
3. Нажмите "Create a new release"
4. В поле "Choose a tag" введите: `v1.0.0` (если тега нет, он создастся автоматически)
5. Заполните:
   - **Release title:** `SafeKey v1.0.0`
   - **Description:** Описание изменений
6. Загрузите файл `SafeKey-Setup-1.0.0-x64.exe` из папки `release/`
7. Нажмите "Publish release"

### Вариант B: Через Git (если установлен)

```bash
# 1. Создать тег
git tag v1.0.0

# 2. Отправить тег на GitHub
git push --tags

# 3. Затем создать Release через веб-интерфейс GitHub
```

## Шаг 3: Проверка автообновления

После публикации релиза:
- Приложение автоматически проверит обновления при запуске
- Или в настройках нажмите "Проверить обновления"

## Важно:

- Версия в `package.json`: `"version": "1.0.0"`
- Тег должен быть: `v1.0.0`
- Название установщика: `SafeKey-Setup-1.0.0-x64.exe`
- Репозиторий должен быть публичным
