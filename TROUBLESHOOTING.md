# Решение проблем SafeKey

## Белый экран при запуске

### Проблема
Приложение открывается, но показывает только белый экран.

### Решение

1. **Пересоберите приложение:**
   ```bash
   npm run build:win
   ```

2. **Проверьте консоль разработчика:**
   - Нажмите `F12` в открытом приложении
   - Или `Ctrl+Shift+I`
   - Посмотрите на ошибки в консоли

3. **Проверьте пути к файлам:**
   - Убедитесь, что папка `dist/renderer/` содержит файлы
   - Проверьте наличие `index.html` и папки `assets/`

4. **Если проблема сохраняется:**
   - Удалите папки `dist/` и `release/`
   - Выполните полную пересборку:
     ```bash
     npm run build
     npm run build:win
     ```

## Другие проблемы

### Приложение не запускается

- Убедитесь, что вы используете правильный исполняемый файл: `release\win-unpacked\SafeKey.exe`
- Проверьте, что все файлы в папке `win-unpacked` на месте
- Попробуйте запустить от имени администратора

### Ошибки в консоли

Откройте консоль разработчика (`F12`) и проверьте ошибки. Частые проблемы:
- Ошибки загрузки модулей
- Проблемы с путями к файлам
- Ошибки инициализации базы данных

### База данных не создается

- Проверьте права доступа к папке `%APPDATA%\SafeKey\`
- Убедитесь, что приложение может создавать файлы в этой папке
