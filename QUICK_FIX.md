# Быстрое исправление белого экрана

## Проблема
Приложение открывается, но показывает белый экран.

## Решение

### Шаг 1: Пересоберите приложение

Выполните в терминале:

```bash
cd E:\Progect\SafeKey
npm run build:win
```

### Шаг 2: Проверьте результат

После сборки проверьте:
1. Папка `dist/renderer/` должна содержать:
   - `index.html`
   - Папку `assets/` с файлами `.js` и `.css`

2. Папка `release/win-unpacked/` должна содержать:
   - `SafeKey.exe`
   - Папку `resources/` с `app.asar`

### Шаг 3: Запустите заново

Запустите `release\win-unpacked\SafeKey.exe`

### Если проблема сохраняется

1. **Откройте консоль разработчика:**
   - Нажмите `F12` в открытом приложении
   - Посмотрите ошибки в консоли

2. **Проверьте пути:**
   - Убедитесь, что файлы в `dist/renderer/` используют относительные пути (например, `./assets/...` вместо `/assets/...`)

3. **Полная пересборка:**
   ```bash
   # Удалите старые сборки
   rmdir /s /q dist
   rmdir /s /q release
   
   # Пересоберите
   npm run build:win
   ```
