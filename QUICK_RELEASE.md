# Быстрая инструкция по созданию релиза

## ✅ Код собран успешно!

## Шаг 1: Создание установщика

1. Откройте файл `installer.iss` в **Inno Setup Compiler**
2. Нажмите **F9** или **Build → Compile**
3. Установщик будет создан: `release\SafeKey-Setup-1.0.0-x64.exe`

## Шаг 2: Создание тега и релиза на GitHub

### Способ 1: Через веб-интерфейс GitHub (рекомендуется)

1. Откройте: https://github.com/Actoric/SafeKey/releases/new
2. В поле **"Choose a tag"** введите: `v1.0.0` (если тега нет, он создастся автоматически)
3. **Release title:** `SafeKey v1.0.0`
4. **Description:** 
   ```markdown
   ## Что нового
   - Исправлена навигация между разделами
   - Добавлена проверка автообновления
   - Улучшен оверлей с раскрытием паролей
   - Добавлен автозапуск и работа с системным треем
   ```
5. Перетащите файл `SafeKey-Setup-1.0.0-x64.exe` в область "Attach binaries"
6. Нажмите **"Publish release"**

### Способ 2: Через Git (если установлен)

```bash
git tag v1.0.0
git push --tags
```

Затем создайте Release через веб-интерфейс GitHub.

## Готово! 

После публикации релиза приложение автоматически найдет обновление.
