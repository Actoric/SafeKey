# Инструкция по созданию установщика SafeKey

## Требования
- Inno Setup (установлен на вашем компьютере)
- Собранная версия приложения в `release\SafeKey-win32-x64\`

## Шаги для создания установщика

### 1. Убедитесь, что приложение собрано
```bash
npm run build
npx electron-packager . SafeKey --platform=win32 --arch=x64 --out=release --overwrite --icon=build/icon.ico
```

### 2. Откройте Inno Setup Compiler
- Запустите Inno Setup Compiler
- Файл → Открыть (File → Open)
- Выберите файл `installer.iss` из корня проекта

### 3. Создайте установщик
- Нажмите кнопку "Build" (Собрать) или нажмите F9
- Дождитесь завершения сборки
- Установщик будет создан в папке `release\SafeKey-Setup-1.0.0-x64.exe`

## Альтернатива: Командная строка

Если Inno Setup установлен в стандартную директорию, можно использовать командную строку:

```bash
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

## Настройки установщика

Установщик будет:
- Устанавливать приложение в `C:\Program Files\SafeKey\`
- Создавать ярлык в меню "Пуск"
- Опционально создавать ярлык на рабочем столе
- Создавать запись в "Установка и удаление программ"
- Поддерживать русский и английский языки

## Результат

После сборки в папке `release\` будет создан файл:
- `SafeKey-Setup-1.0.0-x64.exe` - установщик приложения
